# Goal

Design and implement a Role-Based Access Control (RBAC) system (specifically the Role Management module) for a backend application.


# Instruction

## Backend

- create a role module
- implement role CRUD functionalities
- admin have to send role information (role name) and permissions
- slug should be auto created
- createdBy should be admin or tenant
- type should be role


## Frontend


- there is already add role modal exits. Use this modal for adding
- show all the roles in the roles table
- delete role clicking on delete icon of the table. Make sure to add a confirmation
- edit role clicking on the edit icon. Use the same modal for edit also just change the button name
