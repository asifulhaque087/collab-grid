# Build page and call apis

## Goal

Build listed pages in @apps/web and implement respective apis that are already implement in @apps/api.

- login
- register
- forgot-password
- reset-password

## Note

After logged in user should not access those pages

## Requirements

- login and register both page should include signin with google button.
- login form should also include a forgot password link.
- check apps/api/src/auth/auth.controller.ts apps/api/src/auth/auth.service.ts to understand the required form field for every page.
- make sure login with google button is working

Look at the @apps/api/src/auth folder for apis references.

## Ui

refer http://localhost:3000/ this  landing page for ui. Make sure pages follow the exact same UI/UX design, layout style, and theme as the main landing page so they are visually consistent with the landing page.
